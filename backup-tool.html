1 위 JSON 파일(들)을 다운로드합니다.
2 GitHub 저장소를 만들고 viewer.html과 JSON 파일을 업로드합니다.
3 저장소 Settings → Pages → Branch: main으로 설정합니다.
4 https://[username].github.io/[repo]/viewer.html?file=[파일명].json URL을 공유합니다.
API 키 발급 방법
1 Google Cloud Console 접속 후 프로젝트 생성
2 "Google Sheets API" 검색 후 사용 설정
3 APIs & Services → Credentials → API 키 생성
4 스프레드시트를 "링크 있는 모든 사용자" 공유로 설정
API 키는 이 페이지에서만 사용되며 저장되지 않습니다.
